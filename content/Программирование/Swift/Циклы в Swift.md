## Цикл for in
- Создаётся с помощью ключевого слова for 
- Пишем константы
- Открываем скобку
- Затем пишем диапазон через многоточие 
- Закрываем скобку
```swift
for a in 1...10{
print(a)
}
```
## Цикл while
- Создаётся с помощью ключевого слова while
- Рядом с ним пишем условие 
```swift
var a = 10
while a > -1 {
    print (a)
    a -= 1
}
```
